# MUD (Multi-User Dungeon) Project in C++

This project is a text-based C++ simulator showcasing skills including:

- Object-oriented design
- Modular compilation and linking
- State-driven logic
- Heaps/Stacks
- JSON-based configuration
- Command-line interaction
- Build automation with Make

This codebase was developed in an academic security lab environment and has
been sanitized to remove any challenge-specific data, exploits, or flags.
It is presented here solely to demonstrate software engineering and systems
design skills.
